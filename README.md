# Suproteek Banerjee Sci-Fi Portfolio

Deploy-ready Vite + React portfolio for Suproteek Banerjee.

## Local Development

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
```

## Vercel

Import this folder in Vercel:

```text
/Users/suproteek/Developer/suproteek-sci-fi-portfolio
```

Vercel should detect Vite automatically.

- Build command: `npm run build`
- Output directory: `dist`
- Install command: `npm install`
